# Design notes

The approved direction is **Cinematic Noir**: premium, dark, editorial, entertainment-focused, and mobile-first. Near-black surfaces dominate; refined gold is reserved for meaningful actions and states. Typography, spacing, sharp geometry, media behavior, and accessibility requirements are defined in the Developer Bible.

## Source material

- `stitch-export/` contains the original ZIP and extracted Stitch reference package.
- `references/logo/` contains the supplied logo studies.
- `screenshots/` is reserved for curated design screenshots that are independent of a source export.

Stitch HTML is not production source code. Preserve it for visual comparison only; rebuild approved screens using the project architecture, reusable components, design tokens, semantic HTML, and responsive behavior.

The screenshots currently embedded in the Stitch export remain beside their paired HTML files so the source package stays intact.
